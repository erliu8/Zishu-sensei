================
项目简介
================

什么是 Zishu-Sensei？
======================

Zishu-Sensei（自树先生）是一个先进的 AI 学习助手系统，旨在为学习者提供智能化、个性化的学习支持。
项目名称中的"自树"寓意着自我成长、自主学习的理念。

系统采用最新的人工智能技术，包括大语言模型（LLM）、检索增强生成（RAG）、知识图谱等，
为用户提供全方位的学习辅助功能。

核心功能
========

智能对话
--------

- **自然语言交互**: 支持流畅的中英文对话
- **上下文理解**: 能够理解多轮对话的上下文关系
- **个性化响应**: 根据用户特征调整回答风格

知识检索
--------

- **向量检索**: 基于语义相似度的快速知识查询
- **文档索引**: 自动索引和管理学习资料
- **多源整合**: 整合多个知识源提供全面答案

学习辅助
--------

- **题目解答**: 支持各学科题目的讲解和解答
- **概念解释**: 深入浅出地解释复杂概念
- **学习建议**: 提供个性化的学习路径建议

知识管理
--------

- **笔记整理**: 智能整理和分类学习笔记
- **知识图谱**: 构建学科知识关联网络
- **进度跟踪**: 记录和分析学习进度

技术特点

高性能
------

- 异步 I/O 处理，支持高并发
- 智能缓存策略，减少响应时间
- 数据库查询优化

可扩展
------

- 插件化适配器架构
- 支持动态加载和卸载模块
- 易于集成第三方服务

安全可靠
--------

- JWT 认证和授权

易于部署
--------

- Docker 容器化部署
- 支持云平台和本地部署
- 自动化部署脚本
