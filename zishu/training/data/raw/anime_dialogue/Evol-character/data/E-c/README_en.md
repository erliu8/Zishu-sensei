## Download Datasets


This dataset is generated by GPT3.5 and GPT4. To ensure the responsible use of the data, only a portion of the data has been made public at this time. The publicly available data consists of three files, with each file containing settings and dialogues for 200 characters. You can **download the publicly available data or apply to access the entire dataset** on Hugging Face.: 
[![English](https://img.shields.io/badge/Download-Huggingface-yellow)](https://huggingface.co/datasets/bai-roleplay/evol-character)

Information about the data generation code can be obtained in github: [![English](https://img.shields.io/badge/Code-Github-blue)](https://github.com/Bauhinia-AI/evol-character)

## Data Generation Framework
![Local Image](../main/assets/flowchart.png)

- **Seed Characteristic Set and Base Settings**:
  - A manually written seed set contains basic character traits.
  - The large language model (LLM) generates base settings for characters from this seed set.
- **Evolution of Character Settings**:
  - A second seed set contains instruction prompts that guide the evolution of character settings.
  - These evolve-character instruction prompts are embedded into an instruction pool.
  - The base settings are sampled and evolved through these prompts, facilitated by the LLM, resulting in evolved settings.
- **Feedback Loop and Refinement**:
  - The evolved settings are subject to a mixed evaluation system, which includes both GPT-4 and human reviewers.
  - Feedback from this evaluation is used to iteratively update and refine the seed sets, leading to a polished, fine-grained character setting dataset.
- **Role-Playing and Dialogue Generation**:
  - The refined character settings are then used in a self-instruction framework.
  - This results in the generation of role-playing dialogues between characters and users.


## Data Structure
We have 3 datasets:
- evol-character-gpt3.5.json
- evol-character-male-gpt3.5.json
- evol-character-gpt4.json

We first generated the data `evol-character-gpt3.5.json` in which the majority of the character roles are female. Therefore, we supplemented by generating male character data `evol-character-male-gpt3.5.json`.

1. `evol-character-gpt3.5.json`: This dataset includes 200 distinct characters. The data for each character is divided into two parts: instruction and dialog. ### Our Advantages
- **Refined Character Setting Data**: Our dataset improves on the limited character settings in other open-source role-playing data. It includes extensive details about character identity, speaking style, and backstories. The GPT-4 version also adds settings for the identity of the conversational partner, enriching the data.

- **Diverse Character Personalities**: Our dataset encompasses a wide range of unique character personalities, ensuring diversity and richness.

- **Vivid Language and Action Descriptions**: The dataset contains character dialogues and action descriptions, making interactions more realistic and enhancing the role-playing experience.
  
- **Universal Role-Playing Data Generation Framework**: We offer a framework for generating role-playing data, maximizing the potential of the OpenAI API. This framework, currently under testing and optimization, is designed for data fine-tuning and RAG. It will be released soon.
